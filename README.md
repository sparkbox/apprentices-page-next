# Sparkbox Apprentice Page

## Getting Started

- run the following in your terminal: ```git clone https://github.com/sparkbox/apprentices-page-next.git``` 

##### Installing Deps:

- ```npm install```

##### Running the Project:

- ```npm run dev```

##### Running the linter:

- ```npm run lint```

##### Running Tests with watch mode:

- ```npm test```

##### Runnnig tests without watch mode:

- ```npm test:ci```
